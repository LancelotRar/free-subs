## 🚀 本项目亮点

  -  **永久固定：** 订阅链接长期有效，一次导入，终身自动更新。
  -  **主流支持：** 深度适配 Clash、v2ray、Sing-box 等主流代理客户端。
  -  **CF 赋能：** 基于 Cloudflare 高性能开源方案搭建，兼顾速度与稳定性。
  -  **全协议覆盖：** 提供单节点及全平台订阅格式，灵活切换。
  -  **本项目当前使用Cmliu edgetunnel 搭建**

> 强烈建议在客户端内设置 **“自动更新频率 (Update Interval)”** 为 24 小时，以减少CF Workers请求数，以免刷爆导致订阅不可用。 完全不影响使用。
